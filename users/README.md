# Users

Users application providing support for registering/logging in users

# Prerequisites

In order to build and run the application you need to have Java 1.8 and Maven installed

# Installing

Hit the following command in order to build and install the application:

```
mvn clean install
```

# Running

## Run the following script for starting up the application locally

```
./run_users.sh
```

## Sending user register request locally

```
./send_user_register.sh
```

## Sending user login request locally

```
./send_user_login.sh
```
